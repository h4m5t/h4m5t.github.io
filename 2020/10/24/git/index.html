Git

1.git clone

将远程仓库克隆到本地，包括此仓库的版本变化信息。

不需要进行git init操作

方法：https或SSH

2.git pull

拉取远程更新到本地仓库，相当于从远程仓库或许最新版，然后与本地的分支合并。

git pull = git fetch + git merge
